# Bad Memory Design Assets

Small SVG exports pulled from the current site style for sharing, posts, docs, and production handoff.

## Files

- `bad-memory-logo.svg` - stacked pixel logo with skull mark.
- `skull-icon.svg` - white pixel skull mark.
- `cursor-icon.svg` - white pixel cursor.
- `spark-icon.svg` - yellow plus/spark.
- `floppy-icon.svg` - pink/cyan floppy icon.
- `avatar-wizard.svg` - blue wizard player avatar.
- `avatar-frog.svg` - green frog player avatar.
- `avatar-skull.svg` - purple skull player avatar.
- `avatar-pirate.svg` - pirate player avatar.
- `character-sheet.svg` - simple four-character reference sheet.

## Palette

- Black: `#020202`
- White: `#f1f1ed`
- Pink: `#ec4b9d`
- Yellow: `#ffe43e`
- Green: `#59df50`
- Purple: `#b850ff`
- Cyan: `#50e0ca`

## Type

The site currently uses pixel font families from `@fontsource`: `Press Start 2P`, `Tiny5`, and `VT323`.
